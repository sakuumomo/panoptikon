# Nix flake implementation plan

> Historical plan for the first **dev-shell-only** flake.  
> **Status: completed and extended.** Packaging/module/tests live under `nix/`  
> and are documented in [`nix/README.md`](../../../nix/README.md).

**Original goal:** Root flake with CPU / CUDA / ROCm development shells.  
**Delivered:** Shells + `panoptikon` / `panoptikon-desktop` packages + NixOS
module + package/service checks.

**Spec:** `docs/superpowers/specs/2026-07-16-nix-flake-design.md` (updated)

---

## File map (current)

| File | Responsibility |
| --- | --- |
| `flake.nix` / `flake.lock` | Inputs, overlay, packages, shells, checks |
| `nix/package.nix` | Server package (bundled + UI) + `passthru.tests` |
| `nix/desktop.nix` | Tauri desktop + server sidecar + install smoke |
| `nix/module.nix` | `services.panoptikon` |
| `nix/rocm-packages.nix` | Shared HIP runtime pkgs (module + rocm shell) |
| `nix/tests/panoptikon.nix` | NixOS VM test (cpu) |
| `nix/tests/panoptikon-rocm-config.nix` | NixOS VM test (rocm wiring) |
| `nix/README.md` | Packaging/runtime contract |
| `scripts/generate-nix-dev-config.py` | `nix develop` config (bare tool names) |
| `config/server/nixos.toml` | Service seed config |
| `panoptikon/src/host_paths.rs` | Store-path-free host discovery |
| `panoptikon/src/accelerator_env.rs` | Resolved-accelerator worker env + setup probes |
